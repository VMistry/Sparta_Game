How to Play

  1. To start the game, you would need to press “New Game”. This will start up the function which will unlock buttons for Rock, Paper and Scissor.
  2. Once you have clicked “New Game”, you would then need to select either the Rock, Paper or scissors button.
  3. Once you have chosen a button, the computer will generate its own choice. It would then see who won depending on what both you and the computer has chosen. The winner will gain one point.
  4. To starts the next game, repeat steps 1 to 4.
